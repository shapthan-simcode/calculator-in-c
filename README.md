# Calculator in C

This is a simple Calculator program written in C.  
It performs basic arithmetic operations:  
- Addition (+)  
- Subtraction (-)  
- Multiplication (*)  
- Division (/)

---

## Features
- Menu-driven (choose an operator)  
- Works with floating-point numbers  
- Handles division by zero  

---

1. Clone the repository:
